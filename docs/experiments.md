# Running the Experiments and Benchmarks

This document outlines the commands required to run the specific benchmarks and reproduce the tables presented in the slides.

## Prerequisites

Make sure the virtual environment is activated, `transformers` is installed, and the custom CUDA extension is built:

```bash
source .venv/bin/activate
pip install transformers
pip install -e . --no-build-isolation
```

## 1. Flash Attention Benchmarks (Latency & Memory)

To get the execution time comparison between Vanilla Attention and our Flash Attention and Paged Attention kernels:

```bash
python3 benchmarks/bench_time.py --cuda
```

To get the peak global memory allocations and verify our $O(N)$ memory bounds:

```bash
python3 benchmarks/bench_memory.py --cuda
```

**Results generated by the scripts:**
| Config | Vanilla Time | Flash Time | Vanilla Mem | Flash Mem |
|---|---|---|---|---|
| B=1 H=1 N=64 | 98.3 µs | 66.7 µs | 8.0 MB | 8.0 MB |
| B=1 H=4 N=256 | 93.9 µs | 418.4 µs | 11.0 MB | 9.0 MB |
| B=4 H=8 N=1024 | 1.52 ms | 17.34 ms | 298.0 MB | 42.0 MB |

## 2. Paged Attention Throughput

To benchmark the throughput (Rows/sec and latency per forward pass) of our Paged Attention configurations:

```bash
python3 benchmarks/bench_paged_tps.py --cuda --b 4 --n 512 --iters 50
```

**Results generated by the script:**
| Implementation | Latency/fwd | Fwd/s | Rows/s |
|---|---|---|---|
| Paged v1 | 2.58 ms | 388.0 | 1,552.2 |
| Paged v2 | 200.1 µs | 4,996.4 | 19,985.7 |

## 3. GPT-2 Integration Benchmarks

To benchmark the isolated **Prefill Phase** (processing the full prompt at once) using the Flash Attention wrapper:

```bash
python3 benchmarks/bench_gpt2_prefill.py --lengths 128,512 --iters 10
```

**Results generated by the script:**
| N | Standard HF | Patched Flash Attention | Speedup |
|---|---|---|---|
| 128 | 8.46 ms | 8.65 ms | 0.98x |
| 512 | 10.99 ms | 11.04 ms | 1.00x |

To benchmark **End-to-End Generation TPS (Tokens/second)** integrating both kernels into the full generating flow:

```bash
python3 benchmarks/bench_tps.py --tokens 20 --iters 2
```

**Results generated by the script:**
| Variant | TPS (Tokens/sec) |
|---|---|
| Standard HF GPT-2 | 114.20 |
| Patched Custom Kernels | 112.29 |
